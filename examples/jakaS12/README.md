# Jaka S12 Teleoperation Examples

This directory contains examples for using Jaka S12 robots in teleoperation scenarios, where one robot (the leader) acts as a controller for another robot (the follower).

## Files

- `jakaS12_teleop_example.py`: Main example script demonstrating teleoperation
- `config.py`: Configuration example for leader and follower robots

## Usage

To run the teleoperation example:

```bash
python examples/jakaS12_teleop/jakaS12_teleop_example.py
```

Make sure to update the IP addresses in the configuration to match your robot setup.
